# graphql-sample
## Description
このプロジェクトは、GraphQLのサンプルプロジェクトです。
以下の記事を参考に作成しています。
[Goで学ぶGraphQLサーバーサイド入門](https://zenn.dev/hsaki/books/golang-graphql/viewer/originalserver)<br>

## Usage
### サーバーの起動
```bash
go run server.go
```
サーバーが起動したら、`http://localhost:8080`にアクセスしてください。
